### Customize Clock On Lock Screen - Extension (GNOME V45 and above)

### Install
```
git clone https://github.com/pratap-panabaka/gse-customize-clock-on-lock-screen
cd gse-customize-clock-on-lock-screen
./install.sh
```

### Uninstall
```
./uninstall.sh
```
